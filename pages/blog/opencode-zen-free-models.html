Free AI Models Now Available Through Opencode Zen Gateway

Maurice Wipf · May 11, 2026

Capuzzella now supports Opencode Zen Gateway, giving you access to high-quality AI models from leading Chinese AI labs. Even better — Zen offers free tiers for several powerful models, making AI-powered content creation accessible without a credit card.

What is Opencode Zen Gateway?

Zen Gateway is Opencode's unified API that provides access to state-of-the-art language models from Moonshot AI, Alibaba Cloud, MiniMax, Zhipu AI, and more. Instead of managing multiple API keys and endpoints, Zen gives you one integration point with OpenAI-compatible formatting.

The models available through Zen are highly capable and excel at coding, reasoning, and content generation. Many are open-weight or come from labs with strong track records in AI research.

Setup Guide: Getting Your Opencode API Key

Setting up Zen Gateway in Capuzzella takes just a few minutes. Follow these steps:

Step 1: Create an Opencode Account

  1. Visit opencode.ai
  2. Click Sign Up and create your account
  3. Verify your email address

Step 2: Generate an API Key

  1. Log in to your Opencode dashboard
  2. Navigate to Settings → API Keys
  3. Click Create New Key
  4. Give it a name (e.g., "Capuzzella Site")
  5. Copy the generated key — you'll only see it once!

Step 3: Enter Your API Key in Capuzzella

  1. Open your Capuzzella instance
  2. Click Settings in the navigation bar
  3. Select AI Provider from the menu
  4. Choose Opencode Zen as the provider
  5. Paste your API key in the API Key field
  6. Select a model from the dropdown (choose a -free model to start)
  7. Click Save

That's it! The AI chat panel will now use your selected Zen model. Test it by opening the Editor Panel and sending a message like "Create a hero section for my homepage."

Choosing the Right Model

Not sure which free model to pick? Here's a quick guide:

  • minimax-m2.5-free — Recommended. Best for long documents, summaries, and when you need the largest context window (1 million tokens)
  • nemotron-3-super-free — Great for coding tasks, technical content, and reasoning problems
  • ring-2.6-1t-free — Solid all-rounder with good multilingual support

You can switch models anytime in Settings.
