Getting Started

Welcome! Follow these steps to set up your Capuzzella site and start building.

1

Change Your Password

Your account was created with an initial password. For security, please change it right away in Settings → Account. Choose a strong, unique password that you don't use on other sites.

2

Write Your First Prompt in the Editor Panel

Navigate to any page and append ?edit=true to the URL. The Editor Panel will appear on the right side of the screen.

Describe the changes you want in plain language, click Build page, and watch the AI update your page in real time. Preview with ?draft=true, then publish when you're happy.

3

Customize Components, Design System & CSS Theme

Make the site yours by editing the design system, reusable components, and CSS theme variables.

  • Via the Capuzzella UI — open any page with ?edit=true and use the Editor Panel to update components and styles with AI prompts.
  • Via the Design System — browse /design-system to preview components. Admins and designers can edit component HTML in the browser from the Code button dropdown.
  • Via CLI / Git — edit files directly in the data/ directory and push changes through your normal Git workflow.
4

Add Your Own API Key

AI editing is already included in your subscription, but you can optionally connect your own API key for full control over provider and model choice. Open Settings → AI Providers, choose a provider such as OpenAI or Anthropic, and paste your key. Usage is billed directly by your provider instead of Capuzzella’s managed models.

5

Transactional Emails

Managed transactional email is included in your subscription — sign-in with link, password resets, user invitations, and contact forms work out of the box with no Resend setup required. Open Settings → Transactional Emails to confirm the included sender is active.

Optionally switch to Custom Resend configuration in the same settings page if you want emails sent from your own Resend account, API key, and verified domain.

6

Invite Team Members

Collaborate with your team by inviting them to your Capuzzella instance. Head to Settings → Invite to send invitations. Invited members can edit pages and manage content alongside you.
